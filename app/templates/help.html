Help Section

How everything works?

So here wou will find how to work with every options. Here you will able to know about the features of this website

What is budget?

Through creating budgets you can set a target of expanses. You can take weekly,monthly or yearly expanses target on any perticular category.

Add Category

After you loged in to this website you have to creates some categories of expanses. You can find the Add category section inside home page. You can add categories according to your need

Registeration & Login

  1. Click on register
  2. Register yourself with your details
  3. Go to login page and fill up the same details

How to create Category?

  1. Go to home page then add category section in the top
  2. Enter your category name
  3. Enter any description about that category or leave it empty
  4. Press "+Add" button

Note : you can't create category with same name. Category names must be unique

How to enter a category?

  1. Create categories
  2. Go to Categories section inside home page
  3. Find category according to your need
  4. Press "GoTo >" button to enter that category

How to add expanses?

  1. Enter a category
  2. Go to add new expanse section
  3. Fill details
  4. Press "+Add" button

How to filter expanses according to date?

  1. Chose a date
  2. Press "Sort" button

To see details/Edit expanse/Delete Expanse : Press "Details>" button

How to create budgets?

  1. Go to budget page
  2. Go to the top section(Add New Budget)
  3. Fill details
  4. Press "+Add" button

Feedbacks
